# Book-API

#### Simple client-server project

## Technologies:
- Java 11
- Spring Boot
- Spring Data JPA
- H2 Database
- JsonWebToken
- JavaScript Cookie
- JQuery/Javascript


## Database H2
``` sh
http://127.0.0.1:8080/h2-console/
Password:root
```

## Swgger Documentation
``` sh
http://127.0.0.1:8080/swagger-ui.html
```
